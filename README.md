# Wheel of Destiny Gacha

## Description
Un jeu gacha basé sur le web qui permet aux utilisateurs de tirer des personnages en fonction de leur rareté.

## Fonctionnalités
- Mécanisme de tirage de personnages basé sur la rareté.
- Représentation visuelle des personnages et des packs.
- Interface utilisateur conviviale avec des icônes et des images.
- Affichage des résultats des tirages et gestion d'une collection de personnages.

## Structure du Projet
- **HTML** : `index.html` définit la structure de l'application avec des menus pour le tirage et la collection.
- **CSS** : `styles.css` gère le style visuel de l'application.
- **JavaScript** : 
  - `app.js` gère la logique de navigation et d'interaction.
  - `character.js` définit les personnages disponibles.
  - `rarity.js` définit les raretés et leurs probabilités.
  - `gacha.js` contient la logique de tirage des personnages.
  - `ui.js` gère l'affichage des résultats.

## Instructions d'installation
1. Clonez le dépôt.
2. Ouvrez `index.html` dans un navigateur web.

## Utilisation
- Cliquez sur le bouton de tirage pour sélectionner un personnage aléatoirement.
- Consultez le personnage tiré et ses détails.
- Gérez votre collection de personnages obtenus.
